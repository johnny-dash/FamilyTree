# Installation & Usage

1.  Make sure JDK 8 and Maven 3 are installed
2.  `git clone https://github.com/johnny-dash/family-tree`
3.  `cd family-tree`

## Build and run unit test

`mvn clean install`

## Run

`java -jar target/*.jar`
